Design Principles

80/20 Rule

Good

Reddit: 80% of the content on the website comes from 20% of the users.

Bad

F.C. Kerbeck: The content is so disorganized that the 20% will have trouble using 80% of the website.

Hick's Law

Good

USAA: The dropdowns on the website have headings inside of the dropdown to show categories so the users can easily select the content for them.

Bad

DSR NY: The content on the homepage could be listed in a much more simple way and is unnecessarily complicated.

Chunking

Good

AbsolutePunk: The content is in bite sized pieces. Each story is in the same format so the user knows when that story ends and the next begins.

Bad

Arvanitakis: There is so much going on within the page that I cannot even process what is happening.

Ockham's Razor

Good

Google: The simple design shows the user exactly what they need to see and nothing more.

Bad

The Worlds Worst Website Ever: The design is extremely complicated and everything on the page could be done in a much more elegant way.

Mapping

Good

Michigan State University Underwater Hockey: Each part of the menu is only used for one function, getting to that specific page.

Bad

Pulcinella: None of the pages are mapped well. The user never knows what will appear unless they hover over the arrow and then click it.
